# postgis-fs
PostGIS feature stream, from pnd

## Background
I want to separate feature stream part from [pnd](https://github.com/hfu/pnd)

## Install
```console
$ git clone git@github.com:hfu/postgis-fs.git
$ cd postgis-fs
$ npm install
```

## Usage
```console
$ vi config/default.hjson
$ node index.js
```
